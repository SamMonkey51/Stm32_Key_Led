#### 中断防抖代码
```
uint32_t current_time = HAL_GetTick();  //现在时间

//与上次出发时间间隔
if (GPIO_Pin == GPIO_PIN_3 && (current_time - last_time_button1 > 50)) {  
  //点灯
  HAL_GPIO_TogglePin(GPIOA, GPIO_PIN_6);  
  //重置last_time_button1
  last_time_button1 = current_time;  
}
```
#### 轮询防抖
在点灯前后加上 `HAL_DELAY(20到50);` 即可
